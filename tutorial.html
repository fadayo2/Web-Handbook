Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language, and it’s the backbone of the web. It’s used to structure content for websites so browsers know how to display it. Without HTML, your webpage would just be plain text floating in cyberspace—hardly inspiring!


  • HyperText: The ability to link from one document to another, forming the “web” part of the World Wide Web.
  • Markup Language: A set of tags used to define elements on a page, like “this is a heading” or “this is an image.”

Why HTML?

HTML is essential for creating web pages. It structures the content, defines its purpose, and enables the use of multimedia, links, and forms.



How HTML Works


HTML provides the skeleton of a webpage, which is then styled by CSS and made interactive by JavaScript. Think of it as the foundation of a building—you can’t have a skyscraper without a solid base.


Here’s a basic HTML example:


<!DOCTYPE html>
<html>
<head>
    <title>Introduction to HTML</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>Welcome to the world of HTML. This is your first webpage!</p>
</body>
</html>

Explanation:

  1. <!DOCTYPE html> : Declares the document as HTML5.
  2. <html> : The root element of the HTML document.
  3. <head> : Contains metadata like the page title.
  4. <body> : Contains all the visible content.

Basic Structure of an HTML Document

Every HTML document begins with this line:

<!DOCTYPE html>

It informs the browser you’re using HTML5. Without it, browsers might switch to “quirks mode,” rendering your page inconsistently.


HTML Document Structure

A typical HTML document looks like this:


<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>My HTML Page</title>
</head>
<body>
    <h1>Welcome!</h1>
    <p>This is a sample HTML document.</p>
</body>
</html>
